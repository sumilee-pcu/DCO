## 1. StockBoard란 무엇인가요?

**StockBoard(스톡보드)**는 복잡하고 어려운 주식 정보를 누구나 알기 쉽게, 그리고 한눈에 보기 편하게 제공하는 **주식 대시보드 애플리케이션**입니다. 
토스증권처럼 직관적이고 깔끔한 UI를 지향하며, 데스크탑 모니터는 물론 스마트폰 화면에서도 깨지지 않고 자연스럽게 적응하는 '반응형 웹 디자인(Responsive Design)'이 적용되어 있습니다.

---

## 2. 이 프로젝트의 핵심 기능 💡

* **시장 요약 대시보드:** 코스피, 코스닥부터 나스닥까지 주요 지수의 상태를 시각적인 애니메이션과 함께 직관적으로 띄워줍니다.
* **종목 상세 페이지:** 특정 종목(예: 삼성전자, 애플 등)을 클릭하면 모의 차트와 함께 현재가, 적정 주가, 재무 건강검진 등 상세 지표를 볼 수 있습니다.
* **반응형 다크모드 UI:** 모바일 화면에서는 하단 플로팅 버튼(매수/매도)이 나타나고, PC 화면에서는 넓은 화면을 활용한 3단 그리드가 짜여지는 등 기기에 맞춰 디자인이 변합니다.

---

## 3. 어떤 기술들로 만들어졌나요? 🛠️

이 프로젝트는 최신 유행하는 웹 기술들로 구성되어 있습니다.

* **프론트엔드 (우리가 보는 화면 만들기):**
  * `React (리액트) 18+`: 가장 인기 있는 화면 구성 도구로, 레고 블록 단위로 화면을 조립하게 해줍니다.
  * `Tailwind CSS`: 복잡한 CSS 파일 없이 클래스 이름만으로 디자인을 뚝딱 입힐 수 있는 마법의 도구입니다.
  * `Vite`: 개발 속도를 엄청나게 빠르게 만들어주는 번개 같은 빌드 도구예요.
* **백엔드 (API 없이 동작하는 가벼운 데이터 전달):**
  * `Express (Node.js)`: 화면에 내부적으로 가지고 있는 기본 주식 데이터를 안전하게 전달해 주는 배달부 역할을 합니다.

## 4. 내 컴퓨터에서 직접 실행해보기 (초보자 가이드) 🚀

초보자분들도 아래 명령어를 순서대로 복사해서 터미널(명령 프롬프트)에 붙여넣기만 하면 앱을 바로 켜볼 수 있습니다!

### 1단계: 준비물
컴퓨터에 Node.js 가 설치되어 있어야 합니다. (VS Code 같은 에디터가 있으면 더 좋습니다.)

### 2단계: 설치하기 (단 한 줄!)
다운로드 받은 폴더를 열고(경로 이동 후) 아래 명령어를 입력하여 필요한 부품들을 설치합니다.
```bash
npm install
```

### 3단계: 화면 켜기!
설치가 끝났으면 아래 명령어로 서버와 화면을 동시에 구동합니다.
```bash
npm run dev
```

### 4단계: 감상하기 🎉
웹 브라우저(크롬 등)를 열고 주소창에 `http://localhost:3000` 이라고 입력하세요. 내 손으로 띄운 멋진 주식 앱이 기다리고 있습니다!

## 5. 프로젝트 폴더 구조 파헤치기 📂

코드는 어디에 있을까요? 길을 잃지 않도록 지도를 준비했습니다!

```text
📁 프로젝트 루트
 ├── 📄 server.ts           ➡️ 데이터(모의 데이터)를 제공해주는 가벼운 백엔드 서버 
 ├── 📄 package.json        ➡️ 설치된 패키지 도구 목록
 └── 📁 src/                ➡️ [여기가 핵심] 우리가 눈으로 보는 화면 
      ├── 📄 App.tsx        ➡️ 전체 페이지를 연결해주는 파일
      ├── 📄 index.css      ➡️ 폰트와 다크모드 테마 
      ├── 📁 pages/         ➡️ 구체적인 각각의 화면들 (Market.tsx, StockDetail.tsx 등)
      └── 📁 components/    ➡️ 표, 그래프, 버튼 등 반복해서 쓰는 UI 블록들
```

언제든 코드를 열고 `className`을 고쳐 색상을 바꿔보거나, 화면 배치를 마음껏 수정하며 놀아보세요!
